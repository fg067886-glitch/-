# Apex Pro Tuner

ชุดเครื่องมือปรับแต่งประสิทธิภาพเกมระดับพรีเมียม (Premium Game Enhancement Utility)

## วิธีการทำให้ลิงก์ใช้งานได้บน GitHub (How to Fix White Screen)

หากคุณใช้โปรเจกต์นี้บน GitHub และพบปัญหาหน้าจอขาว ให้ทำตามขั้นตอน "ต้องทำ" ดังนี้ครับ:

1.  **การตั้งค่าใน GitHub (สำคัญมาก):**
    *   ไปที่หน้า Repository ของคุณใน GitHub
    *   คลิกที่ **Settings** (รูปฟันเฟืองด้านบน)
    *   เลือกเมนู **Pages** ทางด้านซ้ายมือ
    *   ในหัวข้อ **Build and deployment** ตรงส่วนของ **Source** ให้เปลี่ยนจาก `Deploy from a branch` เป็น **`GitHub Actions`** (ถ้าไม่เปลี่ยนตรงนี้ หน้าจอจะขาวแน่นอนครับ)
2.  **ตรวจสอบไฟล์:** 
    *   ตรวจสอบว่าในโปรเจกต์ของคุณมีโฟลเดอร์ชื่อ `.github/workflows` และมีไฟล์ `deploy.yml` อยู่ข้างใน (ผมสร้างให้แล้ว)
3.  **การ Push โค้ด:**
    *   เมื่อคุณอัปโหลดโค้ดขึ้น GitHub โปรแกรมจะเริ่ม Build อัตโนมัติ (ดูได้ที่แท็บ **Actions**)
    *   รอจนกว่าแท็บ Actions จะขึ้นเครื่องหมายถูกสีเขียวทั้งหมด แล้วค่อยเปิดลิงก์ครับ

### ข้อมูลสำคัญ
- **รหัสผ่านเข้าหน้าเมนู:** `darkaura`
- **ปุ่ม Authenticate:** เมื่อพิมพ์รหัสเสร็จแล้วให้กดปุ่มนี้เพื่อเข้าสู่ระบบครับ

### หากยังขาวอยู่?
- กดปุ่ม **F12** เพื่อดู Console ถ้าขึ้นว่า 404 แสดงว่าการตั้งค่าในข้อ 1 (GitHub Actions) ยังไม่ถูกต้องครับ
